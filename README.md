# Cache

| ----------------- | ----- | ----- |
| Size of Cache     | 4     | KB    |
| Associativity     | 2-way | -     |
| num of sets       | 128   | -     |
| Blocks/Cache line | 4     | words |
| Address width     | 32    | bits  |
| Data width        | 32    | bits  |
| TAG width         | 23    | bits  |
| index width       | 7     | bits  |
| word  width       | 2     | bits  |
| valid width       | 1     | bit   |
| dirty width       | 1     | bit   |
| LRU width         | 1     | bit   |
